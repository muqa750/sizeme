# 📁 مجلد صور التي شيرت — imagestshirts/

---

## 📐 صيغة اسم الملف

```
{رقم-المنتج}-{رقم-الصورة}-{مفتاح-الصورة}.jpg
```

**مثال فعلي:**
```
01-1-lacoste-e33.jpg   ← الصورة الأولى للمنتج رقم 01
01-2-lacoste-e33.jpg   ← الصورة الثانية لنفس المنتج
01-3-lacoste-e33.jpg   ← الصورة الثالثة (وهكذا)
```

---

## ✅ خطوات إضافة منتج تي شيرت جديد

### الخطوة 1 — سمّ الصورة بشكل صحيح

افرض TOTAL_PRODUCTS الحالي = 29، المنتج الجديد سيكون رقم **30**.

```
30-1-اسم-الماركة.jpg   ← الصورة الأولى (إلزامية)
30-2-اسم-الماركة.jpg   ← الصورة الثانية (اختيارية)
30-3-اسم-الماركة.jpg   ← الصورة الثالثة (اختيارية)
```

> ⚠️ استخدم أحرف إنجليزية صغيرة وشرطة فقط في اسم الملف  
> مثال: `30-1-hugo-boss.jpg` وليس `30-1-Hugo Boss.jpg`

### الخطوة 2 — افتح `index.html` وغيّر رقماً واحداً

ابحث عن:
```js
const TOTAL_PRODUCTS = 29;
```
غيّرها إلى:
```js
const TOTAL_PRODUCTS = 30;
```

### الخطوة 3 — أضف سطراً في `PRODUCTS_META`

ابحث عن قسم التعليقات في آخر PRODUCTS_META وأضف:
```js
30: { sku:'XX-30', imgKey:'hugo-boss', brand:'HUGO BOSS', sub:'Classic Logo',
      colors:[C.BLK, C.WHT, C.RBL] },
```

> ✏️ **شرح الحقول:**
> - `sku` → كود المنتج في رسالة الواتساب (اختر ما تريد)
> - `imgKey` → يجب أن يطابق الجزء الأخير من اسم الصورة  
>   الصورة `30-1-hugo-boss.jpg` → imgKey = `'hugo-boss'`
> - `brand` → اسم الماركة كما يظهر في الموقع
> - `sub` → النص الصغير أسفل الماركة
> - `colors` → الألوان المتاحة (اختر من: C.BLK, C.WHT, C.RBL, C.DNV, C.OLV, C.BRN, C.CHR, C.TPE, C.BRG)
> - `category` → لا تكتبه — التي شيرت هو الافتراضي تلقائياً
> - `catSeq` → لا تكتبه — التي شيرت يستخدم الرقم العام
