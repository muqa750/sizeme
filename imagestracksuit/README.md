# 📁 مجلد صور التراكسوت — imagestracksuit/

---

## 📐 صيغة اسم الملف

```
{catSeq}-{رقم-الصورة}-{مفتاح-الصورة}.jpg
```

**مثال فعلي:**
```
01-1-hermes-ss.jpg   ← الصورة الأولى للتراكسوت رقم 01
01-2-hermes-ss.jpg   ← الصورة الثانية لنفس المنتج
02-1-nike-dri.jpg    ← الصورة الأولى للتراكسوت رقم 02
```

---

## ✅ خطوات إضافة تراكسوت جديد

### الخطوة 1 — سمّ الصور

حالياً عندك تراكسوت واحد (catSeq = 01)، التراكسوت الجديد:

```
02-1-اسم-الماركة.jpg
02-2-اسم-الماركة.jpg
```

### الخطوة 2 — في `index.html` غيّر TOTAL_PRODUCTS

```js
const TOTAL_PRODUCTS = 30;   // كان 29
```

### الخطوة 3 — أضف في PRODUCTS_META

```js
30: { sku:'NK-30', imgKey:'nike-dri', brand:'NIKE', sub:'Dri-FIT',
      colors:[C.BLK, C.DNV, C.RBL],
      category:'tracksuit',
      catSeq:'02',
      added:'2026-04-17' },
```

> ✏️ **الحقول الأساسية:**
> - `category: 'tracksuit'` → يضع المنتج في قسم التراكسوت
> - `catSeq: '02'` → يطابق `02-1-nike-dri.jpg`
> - مقاساته XL تلقائياً (2XL → 7XL) مثل التي شيرت
