# 📁 مجلد صور البولو — imagespolo/

---

## 📐 صيغة اسم الملف

```
{catSeq}-{رقم-الصورة}-{مفتاح-الصورة}.jpg
```

> ⚡ **الفرق عن التي شيرت:** هنا الترقيم مستقل — كل منتج بولو يبدأ من `01`  
> بغض النظر عن رقمه في PRODUCTS_META

**مثال فعلي:**
```
01-1-polo-plus.jpg   ← الصورة الأولى للمنتج البولو رقم 01
01-2-polo-plus.jpg   ← الصورة الثانية لنفس المنتج
02-1-ralph-lauren.jpg ← الصورة الأولى للمنتج البولو رقم 02
```

---

## ✅ خطوات إضافة منتج بولو جديد

### الخطوة 1 — حدّد رقم المنتج الجديد في PRODUCTS_META

حالياً المنتجات: 27 = 01 بولو  
المنتج الجديد سيكون الرقم **30** في PRODUCTS_META (بعد إضافة التالي لرقم TOTAL_PRODUCTS)  
لكن `catSeq` الخاص به في **هذا المجلد** سيكون **02**

### الخطوة 2 — سمّ صور المنتج الجديد

```
02-1-اسم-الماركة.jpg   ← الصورة الأولى (إلزامية)
02-2-اسم-الماركة.jpg   ← الصورة الثانية (اختيارية)
```

### الخطوة 3 — في `index.html` غيّر TOTAL_PRODUCTS

```js
// قبل:
const TOTAL_PRODUCTS = 29;

// بعد إضافة منتج بولو ثانٍ:
const TOTAL_PRODUCTS = 30;
```

### الخطوة 4 — أضف سطراً في PRODUCTS_META

```js
30: { sku:'RL-30', imgKey:'ralph-lauren', brand:'RALPH LAUREN', sub:'Polo Slim',
      colors:[C.BLK, C.WHT, C.RBL],
      category:'polo',
      catSeq:'02',
      added:'2026-04-17' },
```

> ✏️ **الحقل الأهم هنا:**
> - `category: 'polo'` → يخبر الموقع بأنه في قسم البولو
> - `catSeq: '02'` → يطابق اسم الصورة `02-1-ralph-lauren.jpg`
> - `imgKey: 'ralph-lauren'` → يطابق الجزء الأخير من اسم الصورة
