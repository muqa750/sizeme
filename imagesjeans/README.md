البنطرون لا يستخدم نظام XL — يستخدم أرقام الخصر الأوروبية:

| المقاس | الوزن التقريبي |
|--------|---------------|
| 38 | 75 – 85 كغ |
| 40 | 85 – 95 كغ |
| 42 | 95 – 110 كغ |
| 44 | 110 – 125 كغ |
| 46 | 125 – 140 كغ |
| 48 | 140 – 155 كغ |

## 📐 صيغة اسم الملف

```
{catSeq}-{رقم-الصورة}-{مفتاح-الصورة}.jpg
```

**مثال فعلي:**
```
01-1-tranzet-slimfit.jpg   ← الصورة الأولى للبنطرون رقم 01
01-2-tranzet-new.jpg       ← الصورة الثانية لنفس المنتج
02-1-levis-512.jpg         ← الصورة الأولى للبنطرون رقم 02
```

---

## ✅ خطوات إضافة بنطرون جديد

### الخطوة 1 — سمّ الصور

حالياً عندك بنطرون واحد (catSeq = 01)، البنطرون الجديد:

```
02-1-اسم-الماركة.jpg
02-2-اسم-الماركة.jpg
```

### الخطوة 2 — في `index.html` غيّر TOTAL_PRODUCTS

```js
const TOTAL_PRODUCTS = 30;   // كان 29
```

### الخطوة 3 — أضف في PRODUCTS_META

```js
30: { sku:'LV-30', imgKey:'levis-512', brand:"LEVI'S", sub:'512 Slim Taper',
      colors:[C.BLK, C.DNV],
      category:'jeans',
      catSeq:'02',
      added:'2026-04-17' },
```

> ✏️ **تذكر دائماً:**
> - `category: 'jeans'` → يُطبّق مقاسات الأرقام (38, 40...) تلقائياً
> - `catSeq: '02'` → يطابق `02-1-levis-512.jpg`
> - لا تضع مقاسات يدوياً — الكود يتولى ذلك
