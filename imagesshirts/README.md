# 📁 مجلد صور القمصان — imagesshirts/

> ⚠️ **هذا المجلد فارغ حالياً — جاهز لاستقبال أول قميص**

---

## 📐 صيغة اسم الملف

```
{catSeq}-{رقم-الصورة}-{مفتاح-الصورة}.jpg
```

**مثال:**
```
01-1-hugo-boss-shirt.jpg   ← الصورة الأولى للقميص رقم 01
01-2-hugo-boss-shirt.jpg   ← الصورة الثانية
02-1-hermes-shirt.jpg      ← الصورة الأولى للقميص رقم 02
```

---

## ✅ خطوات إضافة أول قميص

### الخطوة 1 — ارفع الصورة باسم صحيح

```
01-1-اسم-الماركة.jpg
01-2-اسم-الماركة.jpg   (اختياري)
```

### الخطوة 2 — في `index.html` غيّر TOTAL_PRODUCTS

```js
const TOTAL_PRODUCTS = 30;   // كان 29
```

### الخطوة 3 — أضف في PRODUCTS_META

```js
30: { sku:'SH-30', imgKey:'hugo-boss-shirt', brand:'HUGO BOSS', sub:'Slim Fit',
      colors:[C.BLK, C.WHT, C.RBL],
      category:'shirt',
      catSeq:'01',
      added:'2026-04-17' },
```

> ✏️ **تذكر:**
> - `category: 'shirt'` → يضع المنتج في قسم القمصان
> - `catSeq: '01'` → يطابق `01-1-hugo-boss-shirt.jpg`
> - مقاساته XL تلقائياً (2XL → 7XL)
